# My Portfolio Dashboard

This is a **portfolio dashboard project** showcasing skills in Python, data analysis, and visualization.

## Features
- Load and analyze company data from CSV
- Summary of revenue, expenses, and new clients
- Visual charts showing revenue vs expenses
- Professional, clean code ready for real-world applications

## Skills Demonstrated
- Python (pandas, matplotlib)
- Data analysis and visualization
- Project organization

## How to Run
1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run `dashboard.py`
